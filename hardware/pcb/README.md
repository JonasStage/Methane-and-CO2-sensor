# PCB files

| File | Description |
|------|-------------|
| `gerber.zip` | Gerber files — upload to JLCPCB (or equivalent) to order the PCB |
| `bom.csv` | Bill of materials with JLCPCB part numbers |
| `cpl.csv` | Component placement list for automated assembly |
| `equipment_list.md` | Full equipment list with prices and supplier links |
