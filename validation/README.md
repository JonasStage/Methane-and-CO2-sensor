# Sensor validation data

Comparison of DIY sensor measurements against a Los Gatos Research (LGR) commercial laser instrument.

| File | Contents |
|------|----------|
| `diffusive_raw_data.csv` | Raw diffusive flux measurements |
| `diffusive_calculated_flux.csv` | Calculated diffusive fluxes |
| `bubbles_raw_data.csv` | Raw ebullitive (bubble) measurements |
| `bubbles_calculated_flux.csv` | Calculated ebullitive fluxes |
| `in-situ_raw_data.csv` | In-situ raw data |

See [Sø et al. (2024)](https://doi.org/10.1029/2024JG008035) for full methodology and results.
