# Required Arduino libraries

Install all libraries below via **Sketch → Include Library → Manage Libraries…** in the Arduino IDE, or download from the links provided.

| Library | Install via Library Manager | GitHub |
|---------|-----------------------------|--------|
| Adafruit ADS1X15 | Search `Adafruit ADS1X15` | [GitHub](https://github.com/adafruit/Adafruit_ADS1X15) |
| Adafruit SHT4x | Search `Adafruit SHT4x Library` | [GitHub](https://github.com/adafruit/Adafruit_SHT4x) |
| Adafruit Unified Sensor | Search `Adafruit Unified Sensor` | [GitHub](https://github.com/adafruit/Adafruit_Sensor) |
| Adafruit BusIO | Search `Adafruit BusIO` | [GitHub](https://github.com/adafruit/Adafruit_BusIO) |
| RTClib | Search `RTClib` | [GitHub](https://github.com/adafruit/RTClib) |
| SdFat | Search `SdFat` | [GitHub](https://github.com/greiman/SdFat) |
| K30 CO₂ I2C | Not in Library Manager — download ZIP | [GitHub](https://github.com/SFeli/K30_CO2_I2C_Arduino) |

> **MiniCore** (board package) is also required. Follow the [MiniCore installation instructions](https://github.com/MCUdude/MiniCore?tab=readme-ov-file#how-to-install).
